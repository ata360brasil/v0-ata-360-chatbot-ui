# ATA360 — Skills para Claude Console

## O que é

14 habilidades customizadas para o Claude Console (console.anthropic.com → Projects → Custom Skills) que dão ao Claude a inteligência interpretativa sobre o domínio jurídico-administrativo brasileiro de contratações públicas.

## Como usar

1. Acesse **console.anthropic.com** → seu projeto
2. Clique em **"+ Adicionar habilidade"**
3. Cole o conteúdo do arquivo `.md` correspondente
4. O campo **"Nome"** = título do arquivo (ex: "Legislação Brasileira")
5. O campo **"Descrição"** = primeira linha após o título
6. O campo **"Instruções"** = seção `## Instruções` completa
